# Linear Algebra For Data Scientist

This repository contains a comprehensive guide to the essential topics of Linear Algebra needed for Data Science and Machine Learning. Each topic is broken down into a separate Jupyter Notebook containing detailed explanations, mathematical foundations, Python implementations, and curated resources.

## Topics Covered

1. **Vectors**: Operations, Norms, Linear Combinations, and Independence.
2. **Matrices**: Definitions, Operations, Special Matrices, Rank, and Inverse.
3. **Systems of Linear Equations**: Representation and Solving Methods.
4. **Matrix Factorizations / Decompositions**: LU, QR, and Cholesky Decompositions.
5. **Eigenvalues and Eigenvectors**: Diagonalization and Applications (PCA, PageRank).
6. **Singular Value Decomposition (SVD)**: Low-rank Approximation and Dimensionality Reduction.
7. **Linear Transformations**: Change of Basis and Neural Network Affine Transformations.
8. **Norms, Distances, and Inner Products**: Vector/Matrix Norms and Cosine Similarity.
9. **Calculus with Matrices (Matrix Calculus)**: Gradients, Jacobians, and Optimization.
10. **Additional Important Topics**: Covariance Matrices, Quadratic Forms, Tensors, and Sparse Matrices.

## Why This Matters for Data Science
- **Data Representation**: Tabular data, images, text embeddings are naturally matrices or vectors.
- **Algorithm Understanding**: Linear regression, PCA, SVD, neural networks are all built on linear algebra.
- **Efficiency**: Numerical linear algebra libraries (BLAS, LAPACK) power Python’s `numpy` and `scikit-learn`.
- **Interpretability**: Concepts like eigenvalues explain variance captured by PCA; singular values reveal data rank.

## Setup
To run the notebooks locally:
1. Clone the repository: `git clone https://github.com/mlnjsh/Linear-Algebra-For-Data-Scientist.git`
2. Install necessary dependencies (e.g., `numpy`, `matplotlib`, `scipy`, `scikit-learn`).

Enjoy learning the mathematical foundations of Data Science!
